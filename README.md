# CRM

This is a simple CRM project

Technologies:

- Express
- React
- Router
- Redux

* Typescript
* Tailwind CSS

Project should have:

- Clock
- Dynamic Title
- Products
- Companies
- Individual clients
- Option to make, generate and print sales offers
- Option to make, generate and print sales orders
